# reparo

config management/automation framework

## Principles

- pip installable
- facilities exposed under `god object`: `j`
- pluggable
- docs and tests are as important as code


## Contribution

- Clean code (pep-8)
- Documentation
- Tests

## Development environment
- clone repo
- `pipenv shell` 

## Running tests
- `make tests`
- `bash runtests.sh`

## Generating docs
- `make docs`
